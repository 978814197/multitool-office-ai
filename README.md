## 项目描述

Multitool Office AI 是一个基于 LangGraph 的多功能智能办公助手 Agent，旨在提升日常办公效率。

### 核心功能

- **资料查询**：智能搜索和检索信息
- **智能聊天**：自然语言交互助手
- **文档处理**：支持各类文档的读取、解析和处理
- **表格处理**：Excel表格文件的 等分析与操作
- **任务清单生成**：对于无法直接处理的任务，自动生成可执行的任务清单
- **代码生成**：根据任务需求生成可处理的代码
- **沙箱执行**：在隔离环境中运行代码，确保主机安全

### 技术特点

- 基于 LangGraph 构建，支持复杂的工作流编排
- 沙箱机制保护主机环境安全
- 模块化设计，易于扩展
